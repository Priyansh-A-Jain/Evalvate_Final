### What you have: STT transcript → Gemini evaluates text quality
### Problem: Gemini cannot hear TONE. It's analyzing words, not HOW they were said.

### ✅ RECOMMENDED: Hume AI (Vocal Emotion + Prosody)

- **What it gives you:** Pitch variation, speaking pace (WPM), volume dynamics, valence, arousal, dominance scores, filler word density
- **How to use:** Send audio segments (per question) to Hume Expression API → get back emotional prosody scores
- **Cost:** 100 mins free, then $0.03/min
- **Link:** https://hume.ai
- **Alternative:** AssemblyAI (has prosody + sentiment + filler word detection, $0.01/min)
- **Alternative:** SpeechBrain (open source, run locally, more setup)

> For "did they speak dull or harsh or soft" — Hume is the only real answer. Nothing else measures this with this accuracy without rule-based logic.

**Real gaze estimation math (replace the bucket logic):**
```python
# iris landmark indices for MediaPipe Face Landmarker
LEFT_IRIS = [468, 469, 470, 471, 472]
RIGHT_IRIS = [473, 474, 475, 476, 477]

def calculate_gaze_score(face_landmarks, frame_width, frame_height):
    """
    Real gaze-toward-camera estimation.
    When person looks at camera, iris center should be at center of eye socket.
    """
    left_iris_center = get_iris_center(face_landmarks, LEFT_IRIS)
    right_iris_center = get_iris_center(face_landmarks, RIGHT_IRIS)
    
    # Eye socket bounding landmarks
    left_eye_inner = face_landmarks[133]   # inner corner
    left_eye_outer = face_landmarks[33]    # outer corner
    right_eye_inner = face_landmarks[362]
    right_eye_outer = face_landmarks[263]
    
    # Normalized iris position within eye socket (0 = far left, 1 = far right)
    left_ratio = (left_iris_center.x - left_eye_outer.x) / (left_eye_inner.x - left_eye_outer.x + 1e-6)
    right_ratio = (right_iris_center.x - right_eye_inner.x) / (right_eye_outer.x - right_eye_inner.x + 1e-6)
    
    avg_ratio = (left_ratio + right_ratio) / 2
    # 0.5 = looking straight at camera, 0 or 1 = looking far sideways
    gaze_score = 1.0 - (2 * abs(avg_ratio - 0.5))  # 1.0 = perfect eye contact
    return max(0.0, min(1.0, gaze_score))
```

## 6. REPORT — What math to use

### Overall Interview Score (weighted):
```
Overall = (
    0.35 × TechnicalScore      # Gemini evaluation rubric (1-10 → normalized)
  + 0.25 × CommunicationScore  # Hume prosody: pace, clarity, confidence
  + 0.20 × EyeContactScore     # Real gaze estimation % of time
  + 0.10 × EmotionScore        # DeepFace: nervousness, confidence ratio
  + 0.10 × StructureScore      # STAR method adherence from Gemini eval
) × 10
```

### Eye Contact Score:
```
EyeContactScore = (frames_with_gaze_score > 0.6) / total_frames × 10
```

### Communication Score (from Hume):
```
CommunicationScore = (
    pace_score          # 120-160 WPM = 10, deviation = -0.5 per 10 WPM
  + clarity_score       # 1 - (filler_word_count / total_words × 10)
  + confidence_score    # Hume dominance score normalized 0-10
  + energy_score        # Hume arousal, penalize monotone
) / 4
```

### Emotion Score:
```
ConfidenceRatio = (happy_frames + neutral_frames) / total_frames
NervousnessRatio = (fear_frames + sad_frames) / total_frames
EmotionScore = (ConfidenceRatio × 10) - (NervousnessRatio × 5), clamped 0-10
```
